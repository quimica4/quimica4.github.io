Atividade 1 --> A

Para ser calor de formação, é preciso que os reagentes sejam substâncias simples. Para ser calor de combustão, a reação deve ocorrer entre algum combustível e o O2, e o valor de ∆H tem que ser negativo, indicando que houve liberação de energia. A única equação termoquímica que atende a todos esses critérios é a II.

Atividade 2 --> A) 2 NH4NO3(s) → 2 N2(g) + O2(g) + 4 H2O(ℓ) ∆H = - 205,6 kJ/mol de NH4NO3(s)

Atividade 2 --> B) HgO(s) → Hg(ℓ) + ½ O2(g) ∆H = + 90 kJ/mol

Atividade 2 --> C) 2 Na(s) + 2 H2O(ℓ) → 2 NaOH + H2(g) ∆H = - 140,9 kJ/mol de Na(s)

Atividade 2 --> D) CO2(g) + H2(g) → CO(g) + 6 H2O(g) ∆H = + 122,8 kJ/mol

Atividade 2 --> E) C4H10(g) + 13/2 O2(g) → 4 CO2(g) + 5 H2O(ℓ) ∆H = - 2,9 kJ/mol

Atividade 2 --> F) HCℓ(g) + H2O(ℓ) → HCℓ(aq) +∆H = - 18 kcal/mol

Atividade 3 --> C

Todas as reações são exotérmicas, liberando calor, pois os valores de ∆H são todos negativos. Mas o que libera mais energia é a reação III.

Atividade 4 --> O reagente que possui maior energia é o I(g), pois a formação do I2(g) é um processo exotérmico, ou seja, que libera energia. Essa é a única reação que possui o ∆H0 negativo.
